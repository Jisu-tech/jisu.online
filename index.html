DSRT GRID — Non-Intrusive Load Monitoring System

A signal intelligence system that disaggregates total electrical consumption from a single main meter into individual appliance-level consumption — no per-appliance sensors required. Reads the orchestra and identifies every instrument.

What I built: Complete end-to-end pipeline from raw electrical signal ingestion through deep learning inference to API serving. Custom event detection module identifies appliance switching events using signal derivative analysis. Built an appliance signature library that captures unique electrical fingerprints across power, harmonics, and transient behavior. Designed a hybrid 1D CNN + Transformer architecture for sequence-to-sequence disaggregation, with an energy conservation constraint built into the loss function ensuring sum of predicted appliance consumptions equals total measured consumption. Implemented Monte Carlo Dropout for uncertainty quantification — every prediction comes with calibrated confidence intervals.

Architecture: 1D CNN + Transformer hybrid · PyTorch · Trained on UK-DALE dataset · FastAPI backend · Streamlit dashboard · Docker deployment

DSRT FAULT — Adversarial-Robust Grid State Estimation

A Graph Attention Network for real-time estimation of complete power grid state — voltage magnitude and phase angle at every bus — from incomplete, noisy, and potentially adversarial sensor measurements. Built to handle cyberattack scenarios where traditional weighted-least-squares state estimators fail.

What I built: Complete grid modeling pipeline using IEEE 14-bus, 30-bus, and 118-bus test systems via PYPOWER simulation. Generated 100K training scenarios covering normal operations, sensor failures, and adversarial false data injection attacks. Built multi-head Graph Attention Network from scratch in PyTorch Geometric — each node aggregates information from connected buses using learned attention weights. Encoded power flow equations and Kirchhoff's laws as physics-informed loss terms, ensuring predictions respect electrical reality. Implemented adversarial training pipeline where attack scenarios are generated dynamically during training to make the model robust to never-before-seen attack vectors.

Architecture: Graph Attention Network · PyTorch Geometric · Physics-Informed Loss · PYPOWER simulation · IEEE test systems · Adversarial training

DSRT SIM — Neural Surrogate for Circuit Simulation

A physics-informed neural network that replaces traditional SPICE-based circuit simulation. Predicts circuit waveforms 1000x faster than the reference simulator while preserving physical accuracy. Generalizes to circuit topologies not seen during training, enabling real-time circuit optimization workflows previously impossible.

What I built: Automated data generation pipeline using NGSpice to simulate thousands of circuit configurations across varying topologies, component values, and operating conditions. Built a Fourier Neural Operator architecture from scratch — chosen because circuit responses live naturally in frequency space. Encoded Kirchhoff's voltage and current laws as soft constraints in the loss function, so the network learns physics not just patterns. Made the entire system end-to-end differentiable, enabling downstream gradient-based circuit optimization — you can backpropagate through the simulator to optimize component values for target specifications. Benchmarked extensively against NGSpice across hundreds of held-out circuit configurations.

Architecture: Fourier Neural Operator · Physics-Informed Neural Network · PyTorch · NGSpice integration · End-to-end differentiable

DSRT INTEL — Multi-Source Technical Intelligence System

An AI system that ingests technical literature, regulatory filings, satellite data, news feeds, and economic indicators to produce structured intelligence assessments on industrial and technical topics. Built for decision-makers who need sourced, citable analysis at scale — not chatbot responses.

What I built: Multi-source ingestion pipeline pulling from ArXiv, NewsAPI, SEC filings, Sentinel Hub satellite data, World Bank economic indicators, and curated technical sources. Built advanced RAG system with semantic chunking, hybrid retrieval (dense + BM25), cross-encoder reranking, and HyDE query transformation. Implemented multi-agent architecture using LangGraph — separate agents for research, analysis, verification, and writing, communicating to produce consensus assessments. Fine-tuned Llama 3.1 8B with QLoRA on technical intelligence report examples so the model learns the specific writing style and citation format. Built knowledge graph layer for temporal reasoning — "how has X evolved over time" queries that pure RAG cannot answer. Every output includes full source citations and confidence scores.

Architecture: Multi-agent system · LangGraph · Fine-tuned Llama 3.1 (QLoRA) · Advanced RAG with reranking · Knowledge graph · Structured output with citations

DSRT WAE — World Analysis Engine

An ambitious early attempt at building a unified analysis engine capable of ingesting diverse global data sources — news, economic indicators, geopolitical events, scientific developments — and producing integrated analytical assessments across domains.

What I built: Designed the conceptual architecture for a multi-modal data fusion system spanning text, structured data, and time series across multiple domains. Built initial data ingestion modules for several source types. Prototyped early analysis workflows. The project's scope ultimately exceeded what the available technical foundation could support — teaching the critical lesson that ambitious cross-domain systems require both broad data engineering depth and specialized ML capability that must be built systematically rather than assembled. WAE's vision lives on in DSRT INTEL, which is the same idea executed properly: narrower scope (technical and industrial intelligence), deeper engineering, fine-tuned models, multi-agent reasoning, and structured outputs with full citation.

Status: Conceptual prototype · Vision evolved into DSRT INTEL · Foundation for multi-source intelligence work
